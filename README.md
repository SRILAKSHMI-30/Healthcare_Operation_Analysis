# 🏥 Healthcare Operation Analysis Dashboard

## 📌 Overview
This project presents a comprehensive **healthcare operations performance analysis** through an interactive dashboard.  
It consolidates metrics such as **revenue, profit, quantity, records, ratings, and operational status**, alongside regional and state-level insights.  
The dashboard is designed to help healthcare administrators and analysts monitor efficiency, identify bottlenecks, and improve service delivery.
